## 🔥 核心功能

**三秒决策卡** —— 打开App前弹出用时时长 + 三个选项（继续 / 限15分钟 / 不刷了）

**替代活动推荐** —— 5大分类25项活动，每项关联真实App一键打开
- 🧘 推荐：5分钟正念呼吸、写深度朋友圈
- 🏃 运动：靠墙静蹲、开合跳、拉伸（每项链接B站跟练视频）
- 📖 阅读：微信读书、维基百科
- 🎨 创意：画画、写日记
- 🎯 学习：多邻国、TED演讲

**社交戒断契约** —— 设定目标邀请朋友互相监督，打卡矩阵可视化

**五层渐进阻断** —— 通知提醒 → 半屏卡片 → 全屏锁页 → 心算解锁 → 永久限额，自由调节

**使用报告** —— 今日饼图 + 本周趋势 + 本月热力图

**软件链接自由配置** —— 22个预设应用 + 自定义，每个替代活动想关联什么App你说了算

## 🚀 快速开始

### Android
下载 `jieduancoach_v3.apk`，直接安装

### iOS（无需审核）
1. 用 Safari 打开部署页面
2. 点击底部 **分享按钮** → **添加到主屏幕**
3. 桌面出现"戒断教练"图标，即可以独立App形式运行

### PC / 浏览器
直接打开 `index.html` 即可体验全部页面

### 启用真实应用打开功能
```bash
python bridge_server.py
```
桥接服务运行在 `localhost:18888`，HTML中的"打开应用"按钮将通过此服务调用真实App
